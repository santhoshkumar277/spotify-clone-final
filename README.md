### 2. Dashboard (Task 2)

### 4. Backend (Task 4)
- **Technology:** Node.js with Express framework.
- **API Endpoints:**
  - `POST /login`: Validates user credentials.
  - `GET /api/songs`: Serves a JSON list of available tracks and metadata.
